About WebAR

WebAR可以轻松地运行在Android、iOS、Windows、Mac系统的Web浏览器上，具有优秀的跨平台性； 模式轻，无需APP，通过浏览器直接开启AR功能，部署方便快捷；开发成本低, URL格式实现链式传播，符合社交媒体信息流动的基本技术要求。

WebAR 可以扫描当前环境中的识别标记，呈现3D模型、动画与真实环境的叠加增强效果。本页面提供一组简单的示例，介绍基于Javascript的增强现实库中的各种特性，包括 AR.js 与 Three.js。查看AR示例需要一个带有摄像头的设备以及一个Hiro或Kanji或LetterA标记图案（点击Examples下方按钮下载图片，打印或显示在第二个设备上）。

AR.js

AR.js 是 Jerome Etienne 开发的一款 Web AR 库，可以用十行 HTML 实现 AR，具有 60+ FPS 的帧率。它主要封装了以下几个库：WebRTC, JSARToolKit, Three.js。

开发框架
技术背景

WebRTC 获取视频流

WebRTC（Web 实时通信，Web Real-Time Communication），是一个支持网页浏览器进行实时语音对话或视频对话的技术。通过WebRTC,可以通过网页唤起用户的摄像头，并且实时获取用户摄像头的图像数据。

JSARToolKit 识别与追踪

JSARToolKit工作原理是跟踪视频流中的特殊标记(AR markers)，JSARToolKit 能够分辨出它们在视频流中的位置以及它们指向哪个方向，通过获得 AR markers 的位置和方向，便可以在影像上正确的画出 3D 对象，让其仿佛就在其中。

Three.js 渲染与交互

Three.js 是基于 WebGL 的渲染库，将繁琐 API 进行了封装优化，开发者可以用可读性更好的代码在前端书写 WebGL，实现虚拟环境及 3D 模型的渲染和显示。
